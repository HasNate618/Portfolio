# Careerly

## Overview
AI-powered virtual job fair built in Unity that makes career exploration fun and inclusive.

## Category
Unity, AI, Education, Career Development

## Status
Hackathon Winner - 4K Smart Projector + MLH Winner Pins, Cohere's Challenge 1st Place (NSBEHacks 2024)

## Tech Stack
- Unity3D
- C#
- Cohere API (AI conversations)
- React (website)
- Express.js
- CSS

## Description

Careerly is a game that simulates a virtual job fair, allowing students to chat with professionals and real-life mentors to learn more about different fields of interest and explore different careers.

### Inspiration

Our inspiration comes from our desire to connect with others to learn more about our interests, build a stable future, and give everyone access to equal opportunities.

### What It Does

- Virtual job fair simulation in Unity
- Chat with AI-powered professional mentors
- Explore different career fields
- Learn about various professions through interaction
- Black excellence representation in important roles (doctors, pilots, artists)

### Key Features

1. **AI-Powered Mentors**
   - Cohere AI integration for realistic conversations
   - Multiple career field representatives
   - Natural language interaction

2. **Diverse Career Areas**
   - Aviation (pilots, flight attendants)
   - Life Sciences (nurses, medical researchers, doctors)
   - Technology (game developers)
   - Arts (musicians)

3. **Unity Environment**
   - 3D virtual job fair space
   - First-person exploration
   - Interactive NPCs

4. **Web Integration**
   - Visually appealing website
   - User input for personalized experience
   - WebGL deployment

### Challenges

- **AI Configuration**: Getting Cohere AI to give accurate, consistent, human-like responses
- **API Integration**: Establishing communication between Unity and Cohere API
- **End-to-End Connection**: Creating solid and secure connection between components

### Accomplishments

- Successfully configured Cohere AI for intended behavior
- Solid end-to-end connection established
- Combined diverse skillsets for exponential productivity
- Created inclusive career exploration tool

### What We Learned

- Unity UI design from scratch
- API communication
- Routing and WebGLs
- First-person game development
- Team collaboration under pressure
