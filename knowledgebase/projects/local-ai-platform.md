# Local AI Platform

## Overview
Fully dockerized local-first AI platform with secure remote access, model routing, MCP tools, and course-specific knowledge bases for grounded study and development workflows.

## Category
AI Systems, Infrastructure, Automation

## Status
Featured Portfolio Project

## Tech Stack
- Docker, Docker Compose
- Linux (Arch/Debian)
- Tailscale (secure networking)
- Open WebUI (chat interface)
- Bifrost (model routing)
- SearXNG (search)
- Crawl4AI (web crawling)
- ComfyUI (image generation)
- MCP Servers (Model Context Protocol)
- llama.cpp (local inference)

## Description

This is my flagship project demonstrating local-first AI infrastructure. It's a complete, self-hosted AI stack that doesn't depend on external APIs for core functionality.

### Key Components

1. **Dockerized Infrastructure**
   - Complete containerized setup for easy deployment
   - Service orchestration with Docker Compose
   - Isolated environments for different AI services

2. **Secure Remote Access**
   - Tailscale integration for secure networking
   - Access from anywhere without exposing ports
   - Zero-trust networking approach

3. **Model Routing**
   - Bifrost for intelligent model selection
   - Hybrid model routing (local + remote)
   - Fallback mechanisms for reliability

4. **MCP Tools**
   - Custom MCP servers for tool-enabled agent workflows
   - Integration with external tools and APIs
   - Extensible tool framework

5. **Knowledge Bases**
   - Course-specific retrieval pipelines
   - PDF parsing and chunking
   - Embedding-based search
   - Grounded responses for study workflows

6. **Search & Crawl**
   - SearXNG for private search
   - Crawl4AI for web content extraction
   - Automated content ingestion

## Architecture
- Containerized microservices architecture
- Local LLM inference with llama.cpp
- Vector database for embeddings
- Web UI for interaction
- API layer for integrations
