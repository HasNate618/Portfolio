# Lumen

## Overview
A calming companion that listens, understands, and responds with personalized mini-games tailored to your emotional state.

## Category
AI, Games, Mental Health

## Status
Hackathon Winner - Best Use of MongoDB Atlas, M5GO IoT Starter Kit (TerraHacks 2025)

## Tech Stack
- React, TypeScript
- Unity3D, C#
- Gemini API
- Phaser.js
- Node.js
- MongoDB
- Clerk (authentication)
- Framer Motion, GSAP

## Description

Lumen is a mental wellness companion that listens when no one else does. It invites users to check in with how they feel through a short guided survey or by journaling freely.

### How It Works

1. User completes emotional check-in or journal entry
2. Gemini analyzes emotional state
3. System offers gentle guidance through personalized minigame
4. Games designed with therapeutic principles to help users feel calmer
5. Early insights indicate potential health issues based on patterns
6. Builds history of emotional trends over time

### Key Features

1. **Emotion Analysis**
   - Gemini interprets nuanced journal inputs
   - Returns emotionally intelligent responses
   - Bridges clinical insight with accessible interface

2. **Personalized Mini-Games**
   - Built in Unity with C#
   - Dynamically adapt to different moods
   - Therapeutic game design principles
   - Embedded in React app with minimal latency

3. **Beautiful UI**
   - Minimal design to avoid overload
   - Satisfying and comforting interface
   - GSAP and Framer Motion animations
   - Intentional spacing and subtle motion

4. **Data Tracking**
   - User emotional history
   - Login frequency and patterns
   - Emotional trend visualization
   - MongoDB for data storage

### Challenges

- **Emotional Intelligence**: Making frontend feel emotionally intelligent without overwhelming user
- **Animation Complexity**: GSAP learning curve and React integration
- **State Management**: Coordinating journal input, emotion detection, and gameplay paths
- **Unity Integration**: Embedding Unity games in React app with clean control handoff
- **Latency**: Structuring backend for fast Gemini processing

### Accomplishments

- Fully working emotional game suite in Unity
- Smooth frontend animations and transitions
- Gemini integration interpreting nuanced inputs
- Polished map experience with multiple features

## Why It Matters
