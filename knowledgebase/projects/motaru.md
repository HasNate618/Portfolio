# Mōtaru

## Overview
Take down ninjas using quick reflexes in this action mobile game.

## Category
Games, Android

## Status
Portfolio Project (Published on Google Play)

## Tech Stack
- Unity3D
- C#
- Google Play

## Description

Mōtaru is an action-packed mobile game where players use quick reflexes to defeat waves of ninjas. Features character customization and multiple levels.

### Gameplay Features

- Fast-paced action combat
- Character customization
- Multiple levels with increasing difficulty
- Reflex-based gameplay
- Score system and leaderboards

### Development

- Built in Unity3D
- C# for game logic and mechanics
- Mobile-optimized controls
- Published on Google Play Store

## Why It Matters
- Demonstrates action game design
- Shows mobile optimization skills
- Evidence of publishing to app stores
- Understanding of game mechanics and balance
