# NowAndThen

## Overview
Geotagged memory map app for leaving comments, photos, and videos tied to locations with AI personality summaries.

## Category
Full-Stack, Mobile, AI, Maps

## Status
Portfolio Project

## Tech Stack
- React Native
- Expo Go
- MongoDB
- Google Maps API
- Node.js, Express

## Description

NowAndThen is a map that reveals nearby memories. Users can leave comments, photos, or videos tied to their exact location, browse what others have left, and replay trails by tracing someone's posts through space and time.

### Key Features

1. **Geotagged Posts**
   - Leave comments, photos, or videos at exact GPS coordinates
   - Browse nearby posts from other users
   - View posts on interactive map

2. **Memory Trails**
   - Trace someone's journey through their posts
   - Replay posts in chronological order
   - Follow paths through space and time

3. **Social Features**
   - Add friends
   - React with likes
   - View friend activity

4. **AI Summaries**
   - AI-generated personality summaries for locations
   - Captures the "vibe" of a place
   - Based on collective user content

5. **Discovery**
   - Radius-controlled comment discovery
   - Browse nearby memories
   - Location-based exploration

### Architecture

- **Frontend**: React Native with interactive map
- **Backend**: Node.js and Express
- **Database**: MongoDB for storing comments and media
- **Maps**: Google Maps API for visualization and location services
- **AI**: Location personality analysis
