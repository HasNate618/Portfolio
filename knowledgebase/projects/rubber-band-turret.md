# Rubber Band Turret

## Overview
Bluetooth-controlled rubber band turret with smooth servo movement and wireless Android app control.

## Category
Hardware, Android, Embedded Systems

## Status
Portfolio Project

## Tech Stack
- Arduino
- C++
- Servo Motors
- Unity (Android app)
- Bluetooth
- Blender (3D design)

## Description

A remotely controlled turret that shoots rubber bands, featuring smooth servo-controlled movement and wireless control via Android app or Bluetooth glove.

### Features

1. **3-Axis Control**
   - Pitch servo (up/down aiming)
   - Yaw servo (left/right aiming)
   - Trigger servo (firing mechanism)

2. **Firing Mechanism**
   - Rotational movement converted to continuous shooting
   - Holds 5 rubber bands before reloading
   - Rapid fire capability

3. **Control Options**
   - Android app built in Unity
   - Bluetooth glove controller (separate project)
   - Smooth servo interpolation

4. **3D Design**
   - Custom designed in Blender
   - Modified models from Thingiverse
   - 3D printed components
   - Optimized for servo integration

### Development

**Mechanical Design:**
- Based on palm-sized gatling gun design
- Modified for servo integration
- Custom barrel with notches for rubber band holding
- 3D printed in multiple iterations

**Electronics:**
- Arduino for servo control
- Bluetooth module for wireless communication
- Power supply for servos
- Circuit design and breadboarding

**Software:**
- Arduino C++ for servo control
- Unity Android app with BLE plugin
- Smooth servo interpolation algorithm
- Bluetooth communication protocol

### Challenges

- **Barrel Design**: Creating notches that hold rubber bands securely while allowing easy release
- **Servo Interpolation**: Learning to make servos move smoothly rather than instantly
- **Mechanism Design**: First time designing a functional mechanism
- **Iteration**: Over 1m tall stack of 3D printed barrels from prototyping

### What I Learned

- Smooth servo interpolation techniques
- 3D design for functional parts
- Bluetooth communication
- Unity mobile app development
- Mechanical design principles
