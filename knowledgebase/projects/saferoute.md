# SafeRoute

## Overview
A real-time hazard reporting app with crowdsourced data, drone integration, and safe routing for Android.

## Category
Android, Full-Stack, Maps

## Status
Portfolio Project

## Tech Stack
- Kotlin
- Google Maps API
- Android Studio

## Description

SafeRoute is a mobile application that helps users navigate safely by reporting and avoiding hazards in real-time. It combines crowdsourced data with intelligent routing to keep users safe.

### Key Features

1. **Real-Time Hazard Reporting**
   - Users report hazards they encounter
   - Categories: construction, accidents, weather, crime, etc.
   - Photo and description support

2. **Crowdsourced Data**
   - Community-driven hazard database
   - Verification system
   - Reputation scoring for reporters

3. **Safe Routing**
   - Google Maps integration
   - Route around reported hazards
   - Alternative path suggestions
   - Safety score for routes

4. **Drone Integration**
   - Aerial view of hazard areas
   - Real-time drone footage
   - Search and rescue support

### Architecture

- **Frontend**: Native Android app in Kotlin
- **Maps**: Google Maps API for visualization and routing
- **Backend**: Cloud-based hazard database
- **Integration**: Drone control APIs

### Challenges

- Real-time data synchronization
- Accurate hazard location mapping
- Drone API integration
- Battery optimization for location services
