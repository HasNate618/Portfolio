# Street Cleaner

## Overview
A mobile game that teaches players what kinds of waste items go in which bins through drag-and-drop gameplay.

## Category
Games, Android, Education

## Status
Hackathon Winner - Second Award (GameHACKS 1.0)

## Tech Stack
- Unity3D
- C#
- Blender (3D models)
- Google Play

## Description

Street Cleaner is a mobile game that promotes cleanliness and proper waste disposal. Players drag litter into the correct bin before time runs out, training memory on waste disposal in a fun way.

### Gameplay

- Litter falls from top of screen
- Players drag items to corresponding waste bins
- Recycling, compost, garbage, and hazardous waste categories
- Timer creates urgency
- End-of-game feedback on performance

### Educational Value

- Players told which items they struggled with most
- Learn proper disposal for each waste type
- Apply knowledge in real life
- Environmental awareness through gameplay

### Development
