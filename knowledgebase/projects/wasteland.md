# Wasteland

## Overview
Horror game with wearable glove controller using accelerometer, flex sensors, and heart rate input.

## Category
Games, Hardware, Wearable Technology

## Status
Hackathon Project (MakeUofT 2025)

## Tech Stack
- Unity3D
- C#
- Arduino
- C++
- Python
- Accelerometer, gyroscope, flex sensors, heart rate monitor
- Bluetooth

## Description

A 3D horror game where player movement and actions are controlled by a wearable glove. The game is set in a maze-like house where the player is trapped with a monstrous entity formed from improperly disposed waste.

### Inspiration

Combining three themes: wearable technology, environmentalism, and video game design. The game teaches recycling through survival horror gameplay.

### Gameplay

- Navigate maze-like house
- Escape from garbage monster
- Recycle waste to craft weapons or keys
- Heart rate affects running speed (higher HR = faster running)
- Complete objectives to escape

### Hardware Controller

**Motion-Sensing Glove Features:**
- Accelerometer for orientation tracking
- Gyroscope for motion detection
- Flex sensors for finger bending
- Heart rate monitor for dynamic gameplay
- Button and joystick inputs
- Bluetooth wireless communication

**Arduino Integration:**
- Arduino Nano with integrated motion sensors
- Bluetooth connectivity
- Real-time sensor data transmission
- Low-latency game control

### Development

**Hardware Side:**
- Arduino programming and circuit design
- Sensor integration and calibration
- Bluetooth communication setup
- Power management

**Software Side:**
- Unity3D environment creation
- Character movement and mechanics
- Monster AI behavior
- Real-time recycling mechanics
- Heart rate-dependent movement speed

### Challenges

- **Hardware Learning Curve**: Two team members new to hardware development
- **Sensor Calibration**: Extensive troubleshooting for accurate readings
- **Limited Components**: Creative solutions for limited jumper wires and sensors
- **Bluetooth Communication**: Stable connection between Arduino and Unity

### Accomplishments

- Functional motion-sensing glove with multiple sensors
- Stable Arduino-Unity communication
- Fully playable 3D game in 24 hours
- Novel heart rate-based gameplay mechanic

### What We Learned

- Circuit design and sensor integration
- Arduino programming fundamentals
- Game development with hardware inputs
- Soldering and breadboard skills
- Creative problem-solving with limited resources
