# Zenith Tower

## Overview
Ascend an endless futuristic dungeon tower in this fast-paced mobile crawler.

## Category
Games, Android

## Status
Portfolio Project (Published on Google Play)

## Tech Stack
- Unity3D
- C#
- Google Play

## Description

Zenith Tower is an endless dungeon crawler set in a futuristic tower. Players fight through increasingly difficult levels, collecting loot and upgrading their character.

### Gameplay Features

- Endless level progression
- Futuristic sci-fi setting
- Fast-paced action combat
- Character upgrades and abilities
- Boss encounters
- Loot collection system

### Development

- Built in Unity3D
- C# for game logic
- Optimized for mobile performance
- Published on Google Play Store
